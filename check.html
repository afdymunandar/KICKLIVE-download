Cara Download & Cara Bermain

Ikuti langkah di bawah ini agar proses download dan instalasi berjalan lancar.

1. Persiapan Sebelum Download

Pastikan perangkat memiliki ruang penyimpanan yang cukup dan koneksi internet yang stabil. Nonaktifkan sementara mode hemat data agar proses verifikasi dan download tidak terputus di tengah jalan.

2. Proses Verifikasi

Setiap link download melewati proses verifikasi otomatis untuk memastikan file yang dibagikan aman dan bebas dari malware. Proses ini hanya memakan waktu beberapa detik dan wajib dilalui sebelum link asli ditampilkan.

3. Cara Install

  • Buka file yang sudah diunduh dari folder Download.
  • Jika muncul peringatan keamanan, pilih "Izinkan dari sumber ini" pada pengaturan perangkat.
  • Tunggu proses instalasi hingga selesai, lalu buka aplikasinya dari menu utama.
  • Jika file berbentuk ZIP/RAR, ekstrak terlebih dahulu sebelum menjalankan file utama.

4. Cara Bermain / Menjalankan

Setelah instalasi selesai, jalankan aplikasi seperti biasa. Berikan izin yang diminta (penyimpanan, koneksi internet, dll) agar seluruh fitur dapat berjalan normal. Jika ada tutorial dalam aplikasi, ikuti panduan tersebut untuk pengalaman terbaik.

5. Mengatasi Error Umum

Jika aplikasi gagal dibuka atau force close, coba restart perangkat, pastikan versi OS memenuhi syarat minimum, dan bebaskan ruang penyimpanan tambahan. Hapus cache aplikasi jika masalah berlanjut.

Tahap verifikasi berikutnya akan mengarahkan Anda ke halaman link download asli.
